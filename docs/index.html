Native Emacs Shell for LLM Agents

A native comint-mode experience for ACP-powered AI agents. Interact with Claude, Gemini, Codex, and more Рђћ all from inside Emacs.

Get Started View on GitHub

Multi-Agent

Claude Code, Gemini CLI, Codex, Goose, Cursor, and 7+ more agents through a unified interface.

Native Shell

Built on Emacs comint-mode. Full keybindings, history, and REPL experience Рђћ no external terminal.

Sessions

Save, resume, and switch between sessions. Persist your work across Emacs restarts.

MCP Support

Model Context Protocol integration for filesystem, HTTP, and custom tool servers.

Containers

Docker and devcontainer support for isolated agent environments.

Rich UI

Syntax-highlighted output, collapsible message blocks, diff viewer, and viewport panel.

Quick Start

;; Install via MELPA
(use-package acp
  :ensure t)

;; Start an agent
M-x acp                          ;; Interactive agent selection
M-x acp-anthropic-start-claude-code  ;; Claude directly
M-x acp-google-start-gemini          ;; Gemini directly
