MELPA Emacs 29.1+

Native Emacs shell to interact with LLM agents powered by ACP (Agent Client Protocol).

Installation

(require 'acp)

Usage

M-x acp

Supported Agents

  • Claude Code (Anthropic)
  • Gemini CLI (Google)
  • Codex (OpenAI)
  • Goose
  • Cursor
  • Qwen Code
  • And more...
