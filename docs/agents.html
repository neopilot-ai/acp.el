Quick Start per Agent

Claude Code (Anthropic)

;; Requires: claude-agent-acp binary
;; Install: npm install -g @anthropic-ai/claude-agent-acp

;; Start directly
M-x acp-anthropic-start-claude-code

;; Or configure as default
(setq acp-preferred-agent-config
      (acp-anthropic-make-claude-code-config))

Adding a New Agent

To add support for a new agent, create a file in agents/:

;;; acp-myagent.el --- MyAgent support for acp -*- lexical-binding: t; -*-

(require 'cl-lib)
(require 'map)

(declare-function acp-make-agent-config "acp")
(declare-function acp--make-acp-client "acp")

(defun acp-myagent-make-config ()
  "Create an agent config for MyAgent."
  (acp-make-agent-config
   :identifier 'myagent
   :mode-line-name "MyAgent"
   :client-maker (lambda (buffer)
                   (acp-myagent-make-client :buffer buffer))
   :authentication-types '(:api-key)))

(defun acp-myagent-make-client (&key buffer)
  "Create a MyAgent ACP client for BUFFER."
  (acp--make-acp-client
   :command "myagent"
   :command-args '("--acp")
   :buffer buffer))

(provide 'acp-myagent)

Then register it in acp.el's acp--make-default-agent-configs.
