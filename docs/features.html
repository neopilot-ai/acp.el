Session Persistence

Sessions can be saved to disk and restored across Emacs restarts.

;; Save current session
M-x acp-session-save

;; Load a session (with completion)
M-x acp-session-load

;; List all saved sessions
M-x acp-session-list

;; Search sessions
M-x acp-session-search RET debug RET

;; Tag a session
M-x acp-session-tag RET my-feature RET

Sessions are stored in <project>/.acp/sessions/ as Emacs Lisp or JSON files.

Permission Manager

Fine-grained control over agent file access and command execution.

;; Check permissions programmatically
(acp-permissions-check-file-read "/path/to/file")
(acp-permissions-check-file-write "/path/to/file")
(acp-permissions-check-command "rm -rf /")

;; View audit log
M-x acp-permissions-view-audit-log

;; Clear audit log
M-x acp-permissions-clear-audit-log

Transcripts

Every session automatically saves a Markdown transcript:

;; Default location
<project>/.acp/transcripts/YYYY-MM-DD-HH-MM-SS.md

;; Contains:
;; - Agent responses with syntax highlighting
;; - Tool calls and results
;; - User prompts
;; - Timestamps

Viewport Panel

Collapsible message blocks for long agent responses. Navigate with TAB and S-TAB.

Diff Viewer

When agents propose file edits, acp.el shows a side-by-side diff view with accept/reject options.
